# ZahadaZtraceneSkoly_workflow

V tomto projektu půjdeme postupně od snazších variant až po tu širokou.
Nejdříve vyzkoušejte vstupní projekt "Útěk ze školy", který je popsán v 'utek_ze_skoly_zadani.md'

Vyzkoušejte si hotovou hrubou aplikaci bez testování, ať máte představu, co vlastně má být cílem.
Stačí si stáhnout 'UtekZeSkoly.zip', rozbalit a spustit exe soubor - otevře se vám terminál
a můžete začít hrát.

Následně budeme programovat náročnější verzi - Záhada ztracené školy.
